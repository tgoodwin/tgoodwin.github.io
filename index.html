INFORMATION

This site features a selection of projects and experiments that I’ve worked on over the past few years.

About myself – I am currently a software engineer based in NYC. I recently finished my undergrad in computer science at Columbia. My academic interests are generally focused on networking, distributed systems, and languages. I am particularly interested in the cloud and notions of real time. I am also passionate about music, analog technology, and most things in between.

I am one of four in Gold Medal, a special music project that began in Fall 2016, and put an album out with Like Young Records in August 2017.

A simple video chat application for the performance evaluation of the WebRTC protocol across various network environments. Featured in IFIP Performance '17.

Blur

Fall 2016

A simple programming language for fast image manipulation and ASCII art creation. The Blur language provides a set of familiar and intuitive syntactic conventions for handling pixel data, useful in image processing algorithms, image to ASCII conversion schemes, and algorithmic/generative ASCII artworks.

Ad Tracker API

Jul 2016

An API built upon the ‘pi-hole’ adblocker project. Monitors DNS traffic leaving a local network to produce information on the advertisement systems connected to a user's Internet activity.

AMuSe Emulator

Jan 2016

AMuSe Emlator, built at Columbia’s Wireless & Mobile Networking Lab, is a web-based visual simulation of the lab’s Adaptive Multicast Services project. The Adaptive Multicast Services project is a WiFi transmission technology that provides a scheme to intelligently process receiver feedback for network adaptation (transmission rate) based on this feedback.
